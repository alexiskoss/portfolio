Petfinder API /

INFO 343 independent project

JavaScript, React, CSS & HTML

Autumn 2016. The goal of the project was to use an API of my choosing to create some sort of interactive website (Find Your Purrfect Match) that used React. I chose to use the Pet Finder API so I created a website that allows users to search for adopatable cats in the area of their choosing. They can search by breed, size, age, location, and various other factors. I also coded a comparison feature that allows users to compare two different cats together so they can quickly see the differences between the two cats that they are looking at. I broke the components of the website up by using React and coded the interactive features with Javascript. I then styled and constructed the website by using CSS and HTML.
